# note 株式マーケットレポート 自動生成ツール

テーマ：「寝かしつけ前のパパ投資家」

実際に保有している米国株・ETF・日本株をベースに、
毎日（簡易版）・毎週（詳細版）でnote記事の「下書き」を自動生成し、メールに送るツールです。

## テーマ・ペルソナについて
- 想定読者は「子育て中で忙しいが、米国株・グロース株投資に興味がある会社員パパ・ママ」。
- 文体の雰囲気は「高校生でもわかる米国株（花子さん）」の初心者向けのわかりやすさ、
  「とも米国株投資チャンネル」のテンポの良さを参考にしていますが、実在の発信者の文章をそのまま
  模倣するものではありません（著作権保護のため、生成AIには「雰囲気の参考程度」と指示しています）。
